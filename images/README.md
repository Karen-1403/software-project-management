## Required Images for the Project

Add the following diagrams and images to this folder:

### Mandatory Diagrams

1. **university-logo.png** (or .jpg)
   - Your university official logo
   - High resolution (at least 300 DPI)

2. **project-lifecycle.pdf**
   - Project phases diagram showing the 6 major phases
   - Create using MS Visio or similar tool

3. **wbs-diagram.pdf**
   - Work Breakdown Structure to 3rd level
   - Hierarchical tree diagram
   - Create using MS Visio

4. **gantt-chart.pdf**
   - Complete project Gantt chart
   - Export from Microsoft Project
   - Should show all activities, dependencies, milestones, and critical path

5. **aon-diagram.pdf**
   - Activity-on-Node network diagram
   - Show all activities with dependencies
   - Highlight critical path
   - Include resource assignments
   - Create using MS Visio

6. **cash-flow-chart.pdf**
   - Line graph showing cumulative cash flow over project lifetime
   - X-axis: Years (0-5)
   - Y-axis: Cumulative cash flow (EGP)
   - Clearly mark the payback period

7. **risk-matrix.pdf**
   - 5x5 probability-impact matrix
   - Color-coded risk zones (green, yellow, orange, red)
   - Plot all identified project risks
   - Create using MS Visio or PowerPoint

### Image Guidelines

- **Format**: PDF for diagrams (vector graphics), PNG/JPG for photos
- **Resolution**: Minimum 300 DPI for raster images
- **Size**: Ensure diagrams fit within page margins when included
- **Naming**: Use lowercase with hyphens (kebab-case)
- **Quality**: Professional appearance, clear labels, readable text

### Tools for Creating Diagrams

- Microsoft Visio (recommended for technical diagrams)
- Microsoft Project (for Gantt charts)
- Draw.io / diagrams.net (free alternative)
- Lucidchart (online tool)
- TikZ in LaTeX (for advanced users)

### Export Settings

When exporting from Visio or other tools:
- Choose PDF format for best quality
- Ensure no compression
- Use A4 or Letter page size
- Landscape orientation for wide diagrams
- Include all layers and elements
