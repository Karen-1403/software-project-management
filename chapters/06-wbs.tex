\chapter{Work Breakdown Structure (WBS)}

\vspace{12pt}

\noindent This chapter presents the Work Breakdown Structure (WBS) for the Online Course Registration Portal project, decomposed to the third level as required. The WBS provides a hierarchical breakdown of all project work into manageable components.

% ============================================================================
\section{WBS Overview}

\par The Work Breakdown Structure organizes project deliverables and work into smaller, more manageable components. Each level provides increasing detail about the work required to complete the project successfully.

\subsection{WBS Principles}

\begin{itemize}[leftmargin=*]
    \item 100\% Rule: WBS includes 100\% of the work defined by project scope
    \item Mutually Exclusive: No overlap between WBS elements at the same level
    \item Outcomes-Oriented: Focus on deliverables, not activities
    \item Three levels of decomposition as specified in requirements
    \item Each work package should be assignable and estimable
\end{itemize}
